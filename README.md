# AI Blog SaaS

## Project Structure
- `app/`: Backend (FastAPI)
- `frontend/`: Frontend (Next.js)
- `database/`: SQL Scripts
- `scripts/`: Utilities

## Setup

### Backend
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the server:
   ```bash
   uvicorn app.main:app --reload
   ```

### Frontend
1. Navigate to frontend:
   ```bash
   cd frontend
   ```
2. Install dependencies (already installed):
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
